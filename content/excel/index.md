Excel remains one of the most widely-used data analysis tools in business. Being an Excel expert is still a highly valued skill.

## 🔑 Core Excel Skills for Data Analysts

| Skill | Why It Matters |
|-------|---------------|
| Pivot Tables | Summarize 100K rows in seconds |
| VLOOKUP/XLOOKUP | Join data from different sheets/files |
| SUMIFS/COUNTIFS | Conditional aggregations |
| Power Query | ETL within Excel |
| Charts & Dashboards | Communicate insights visually |
| Conditional Formatting | Highlight patterns instantly |
| Data Validation | Ensure clean data entry |

## ⌨️ Essential Shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl + T` | Create table |
| `Ctrl + Shift + L` | Toggle filters |
| `Alt + F1` | Create chart |
| `F11` | Create chart in new sheet |
| `Ctrl + ;` | Insert today's date |
| `F4` | Lock cell reference (absolute) |
| `Ctrl + D` | Fill down |
| `Ctrl + R` | Fill right |
| `Alt + =` | AutoSum |

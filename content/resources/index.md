# Learning Resources

Curated resources for data analysts and data scientists. Prioritized for quality — no filler.

---

## 📚 Books

### SQL
- **"Learning SQL"** by Alan Beaulieu — best beginner SQL book
- **"SQL Antipatterns"** by Bill Karwin — avoid common mistakes
- **"T-SQL Fundamentals"** by Itzik Ben-Gan — deep SQL knowledge

### Python & Data Analysis
- **"Python for Data Analysis"** by Wes McKinney (Pandas creator) — essential
- **"Hands-On Machine Learning"** by Aurélien Géron — best ML book
- **"Python Data Science Handbook"** by Jake VanderPlas — free online!

### Statistics
- **"Naked Statistics"** by Charles Wheelan — intuitive intro
- **"Statistics"** by Freedman, Pisani, Purves — rigorous foundation
- **"Thinking Statistically"** by Uri Bram — short but insightful

### Data Storytelling
- **"Storytelling with Data"** by Cole Nussbaumer Knaflic — must read
- **"How Charts Lie"** by Alberto Cairo — avoid misleading visualizations
- **"The Functional Art"** by Alberto Cairo — deeper data viz theory

## 🎓 Courses

### Free
- **Khan Academy Statistics** — excellent probability & statistics foundation
- **Google Data Analytics Certificate** (Coursera) — structured beginner path
- **Kaggle Learn** — free micro-courses: SQL, Pandas, ML, Deep Learning
- **Mode SQL Tutorial** — practical SQL for analytics
- **StatQuest with Josh Starmer** (YouTube) — best stats/ML explanations

### Paid (Worth It)
- **365 Data Science** — comprehensive data science curriculum
- **DataCamp** — hands-on Python, SQL, Power BI, Tableau
- **Udemy: Python Bootcamp** by Dr. Angela Yu — best Python fundamentals
- **Coursera Machine Learning Specialization** by Andrew Ng — industry standard

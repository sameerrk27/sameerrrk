## 🛠️ Built With

[Quartz v5](https://quartz.jzhao.xyz/) — a static site generator for digital gardens.

## 🚀 Local Development

```bash
npm install
npm run install-plugins
npx quartz build --serve
```

Then open http://localhost:8080

## 📝 Adding Notes

1. Create a new `.md` file in the `content/` directory
2. Add frontmatter:
```yaml
---
title: My Note Title
tags:
  - tag1
  - tag2
---
```
3. Write your content in Markdown
4. Link to other notes with `[[note-name|Display Text]]`
5. Commit and push — GitHub Actions will auto-deploy!
